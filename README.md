# Color Adjuster
====

カラーコードを分解して並べ替えたりします

## Description
- カラーコードを入力すると背景色に反映します
- 入力の際は#はつけないでね
- ラジオボタンで選択した色に寄せます（RGB値を入れ替えます）
  - Orange (R > G > B)
  - Pink (R >  B > G)
  - Purple (B > R > G)
  - Azure (B > G > R)
  - Seagreen (G > B > R)
  - Chartreuse (G > R > B)
- Randomボタンでランダムに6色設定します
